# DSA Visualizer

**DSA Visualizer** is a Flask web app that helps users learn Data Structures and Algorithms through interactive visualizations and real-time pseudocode.

## Features
- Visualize sorting, graph, tree, and recursion algorithms step-by-step  
- Play, pause, and control animation speed  
- Responsive, mobile-friendly interface  
- Simple REST API backend generating visualization steps  

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript, Bootstrap, Canvas API  
- Backend: Python, Flask
